读取文件中的内容，将其转为 SQL 插入语句。

**[选项]**

- `-f 文件名.扩展名`，其中文件名必须为表名！

**[注意]**

文件内容格式示例：

+-----------+---------+------------------------------------+--------+
| player_id | team_id | player_name                 | height |
+-----------+---------+------------------------------------+--------+
|     10001 |    1001 | 韦恩-艾灵顿                      |   1.93 |
|     10037 |    1002 | 伊凯·阿尼博古                  |   2.08 |
+-----------+---------+------------------------------------+--------+
